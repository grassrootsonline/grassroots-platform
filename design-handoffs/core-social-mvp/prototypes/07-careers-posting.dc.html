Founding full-stack engineer

Engineering Remote Full-time

We're looking for our first engineering hire — someone who wants to shape both the product and the culture from the ground up. You'll work directly with the founders across the whole stack, from database schema to the details of the feed.

What you'll do

  • Ship features end-to-end across the web app, from data model to UI
  • Help set the technical direction and the bar for quality
  • Work closely with design to build interfaces that feel considered
  • Care about the community we're building for, not just the code

What we're looking for

  • Strong experience with modern full-stack web development
  • Comfort with ambiguity and a bias toward shipping
  • Thoughtfulness about craft, performance, and accessibility
  • Genuine interest in what Grassroots is trying to be

Details

Fully remote. Competitive early-stage compensation and meaningful equity. We hire slowly and treat the process as a two-way conversation.
